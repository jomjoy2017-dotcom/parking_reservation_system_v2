# ระบบจองที่จอดรถ (Parking Reservation System)

ระบบจัดการจอดรถเต็มรูปแบบ รองรับการกรอกข้อมูลรถยนต์ เงื่อนไขเวลาจอด พร้อมคำนวณเวลาและแสดงสถานะอัตโนมัติ (ไฟเขียว/ไฟแดง) รองรับระบบสมาชิก การอัพโหลดไฟล์ และพิมพ์รายงาน PDF ภาษาไทย

## Features
- กำหนดเวลาจอดรถเป็นนาทีและชั่วโมงได้
- กรอกและจัดการหมายเลขช่องจอด ชื่อเจ้าของรถ เบอร์โทรศัพท์ และราคาต่อครั้ง
- เลือกระบุประเภทรถยนต์ (รถยนต์, มอเตอร์ไซค์, รถเก๋ง, รถตู้, อื่นๆ)
- จัดการข้อมูลทะเบียนรถ เพิ่ม แก้ไข และลบข้อมูลได้ (CRUD)
- แสดงสถานะไฟสีเขียวเมื่อกำลังจอด และไฟสีแดงเมื่อหมดเวลาอ้างอิงตามเวลาสากล
- สร้างและปริ้นรายงาน PDF ภาษาไทย สรุปรายวัน เดือน ปี
- ระบบค้นหาข้อมูลจากทะเบียนรถ หรือชื่อเจ้าของรถ
- ระบบสมาชิกและแอดมิน (Login/Register) รองรับมือถือและแท็บเล็ต
- รองรับการอัพโหลดรูปภาพหรือวิดีโอ
